README FOR WINDOWS:
- WIN->cmd->open command prompt
- type java -version->make sure java is installed
- navigate to folder, ex. C:\Users\Tom\312\finalProject -> cd finalProject
- compile project by javac *.java
- to run type "java cachesimulator input.txt" (or a path to the input file)

README FOR MAC:
- finder->go->applications->utilities->open terminal
- when prompted by name$ type java -version, ensure correct version of java
- navigate to correct folder using cd, ex. cd 312/finalProject 
- to compile type java *.java
- to run type "java cachesimulator input.txt" (or a path to the input file)
